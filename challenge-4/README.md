# Challenge 4 - API Integration

## Goal

Using no framework, fetch and display data from the Faker (API)[https://fakerapi.it/]. You may modify `src/style.css`, `src/main.ts`, and `index.html` to achieve this goal.

## Judging Criteria

- Successful API Integration: 50%
- Visual Clarity and Presentation: 30%
- Cleanliness and structure of HTML, Typescript, and CSS: 20%
